## Dependency Management (uv)

This project now uses uv instead of requirements.txt.

### Install dependencies

```bash
uv sync
```

### Run the API server

```bash
uv run python run.py
```

### Run the database sync script

```bash
uv run python syncdb.py
```

### Run tests

```bash
uv run pytest
```
